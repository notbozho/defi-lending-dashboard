# DeFi Lending Dashboard

A modern DeFi lending dashboard built with Next.js 15, Tailwind and Aave SDK. Featuring smooth micro interactions, loading states, wallet integration via wagmi and real-time market data.

## Stack

- Next.js 15 (React 19, Turbopack)
- Tailwind CSS + shadcn/ui
- Zustand
- React Query
- wagmi + RainbowKit
- Framer Motion
- Recharts
- Aave SDK
